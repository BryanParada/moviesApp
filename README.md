## Features 
- Search movies
- View details about a movie
- Watch youtube trailer

## What did i use in this app?
- REST services Consumption
- JSON Conversion --> Maps --> Models
- PageViews
- ApiKeys
- providers
- streams
- Infinite Horizontal Scroll
- SearchDelegate
